# 数据存储

本系统不依赖数据库，以 Shapefile 文件和文件系统作为数据存储。

---

## 1. 数据目录

```
数据根目录 (用户连接)
├── 空间数据/
│   ├── 0工作底图/ (居民点/道路/水系/界限)
│   ├── 2基础数据/ (DEM/土地利用)
│   └── 6指标因子/ (NDVI/土壤湿度/土壤类型/植被类型/植被覆盖度)
│
├── 属性数据/ (自动生成，镜像目录结构)
│   └── 6指标因子/NDVI/ndvi.csv
│
└── 输出目录/ (默认 D:\GeoHazardOutput)
    ├── 土壤类型分析_20260815/ (CSV+JSON+HTML+PNG)
    └── Maps/ (专题图PNG)
```

## 2. Shapefile 结构

每个图层由多个文件组成，系统通过 ArcGIS Runtime 读取 .shp (主文件+关联DBF)。

| 文件 | 内容 | 读取方式 |
|------|------|---------|
| .shp | 几何 | ShapefileFeatureTable.OpenAsync |
| .dbf | 属性表 | 同上，自动关联 |
| .shx | 空间索引 | ArcGIS Runtime 内部 |
| .prj | 投影 | ArcGIS Runtime 内部 |

## 3. 字段类型

| 来源 | ArcGIS类型 | 系统处理 | 示例 |
|------|-----------|---------|------|
| DBF字符 | Text | string | 土壤名称 |
| DBF整型 | Int16/32 | int | FID |
| DBF浮点 | Float32/64 | double | NDVI值 |
| DBF日期 | Date | DateTime | 采集日期 |
| DBF逻辑 | OID | int | 对象ID |

数值型字段 (Int16/32, Float32/64) 可进行分级分区。文本型字段直接按值分组。

## 4. 内存数据模型

### MapLayer

```
LayerId      string          GUID
Name         string          文件名
FilePath     string          完整路径
Type         enum            Vector/Raster
IsVisible    bool
VectorSymbol VectorSymbol    矢量符号(面/线/点)
RasterSymbol RasterSymbol    栅格符号(色带)
Symbols      list            符号预览子项
```

### VectorSymbol

```
FillColor     string    #1565C0
FillOpacity   double    0-1
StrokeColor   string    #FFFFFF
StrokeWidth   double    1.0
PointColor    string    #E81123
PointSize     double    8
```

### RasterSymbol

```
Stops: list of {Color: string, Value: double}
```

### StatResult (分析结果)

```
ParameterName   string    指标名
ClassName       string    分类名
ClassAreaKm2    double    面积(km2)
HazardCount     int       灾害点数
HazardAreaKm2   double    灾害面积
Density         double    密度
Percentage      double    占比(%)
AreaPercentage  double    面积占比(%)
PPa             double    条件概率
CF              double    确定性系数[-1,1]
```

### AnalysisConfig

```
ModuleName       string
OutputFolder     string
ModelMethod      string    CF值法/信息量法
Parameters       list
```

## 5. 输出文件格式

CSV 统计表:
```csv
指标参数,分级分类,面积(km2),灾害点数,密度,占比(%),CF值,易发性评价
土壤湿度,湿润,45.20,23,0.51,15.3,0.3245,高易发
```

CF分级汇总表:
```csv
指标参数,CF均值,CF最大,CF最小,极低区,低区,中区,高区,极高区,灾害总数
土壤湿度,0.1245,0.6234,-0.5432,1,2,3,2,1,150
```

JSON 元数据: 包含模块名、时间戳、源文件路径、总面积、总灾害点、各参数统计

HTML 报告: 表格化的完整分析报告，含CF值和易发性彩色标记

统计图PNG: 柱状图、CF分布图、易发性饼图、综合图

## 6. 数据约束

- 空间数据文件夹只读，分析过程不修改原始SHP
- 属性编辑通过 ArcGIS Runtime UpdateFeatureAsync 写回SHP的DBF
- 每次分析输出独立时间戳目录，不覆盖历史
- CSV编码 UTF-8 with BOM (兼容Excel中文)
- 临时底图文件生成后即清理
