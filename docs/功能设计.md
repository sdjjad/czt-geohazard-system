# 功能设计

截图放在 `screenshots/` 目录下，命名对应即可显示。

---

## 1. 启动界面

程序启动时显示品牌启动屏，3.2 秒后自动进入主界面。深色背景，居中显示系统名称，底部三个加载点动画。

![](data:image/png;base64,<!-- 截图后运行: certutil -encode 01-splash.png tmp.b64 && type tmp.b64 | clip -->)

## 2. 主界面

Ribbon 工具栏在上，左侧数据面板，中间地图，右侧图层面板+符号面板+地理处理面板，底部状态栏。

![](screenshots/02-main.png)

## 3. 数据管理面板

左侧面板，连接本地空间数据文件夹后展示目录树。SHP/TIF 文件高亮显示，双击添加到地图。

操作：连接文件夹 → 目录树浏览 → 双击加载 → 右键菜单

右键菜单：添加到地图、缩放至图层、属性浏览、属性

![](screenshots/03-data-panel.png)

## 4. 图层面板

右侧面板，管理已加载图层。每个图层一行：复选框、名称、符号预览色块。

- 复选框切换可见性
- 点击名称或色块 → 打开符号面板
- 双击名称 → 缩放至图层
- 右键菜单 → 缩放/符号/属性浏览/排序/移除
- 拖拽排序

悬停浅蓝底(#BBDEFB)，选中深蓝边框(#0D47A1)。

![](screenshots/04-layer-panel.png)

## 5. 符号系统面板

右侧面板，编辑图层渲染样式。

**单一符号模式：** 面图层可调填充色、透明度、轮廓色、线宽。顶部预览区实时更新。点色块弹出20色预设色板。

**按字段配色模式：** 选字段 → 选色带 → 应用配色。文本字段生成唯一值渲染，数值字段生成分级渲染。

![](screenshots/05-symbol.png)

## 6. 地理分析面板（土壤植被）

右侧面板，五个分析模块共用：土壤类型、土壤湿度、植被类型、植被覆盖度、NDVI。

操作流程：
1. 选择分类面图层
2. 选择灾害点图层
3. 选择分类字段（实时从SHP读取，不依赖CSV）
4. 可选：选择分级分区方法 + 分类数
5. 可选：选择分析方法（CF值法/信息量法）
6. 运行分析
7. 查看结果：CF值统计表 + 柱状图 + CF分级汇总

分级分区方法：自然间断点、等间距、分位数、标准差，分类数 3-10 可选。

保存结果输出：CSV统计表、CF分级汇总、JSON元数据、HTML报告、4张统计图PNG。

![](screenshots/06-geo-panel.png)

## 7. 属性浏览

选图层 → 显示属性表 → 复制/导出CSV。支持从数据面板和图层面板右键菜单直接打开。

![](screenshots/07-attr-browse.png)

## 8. 属性编辑

选图层 → 可编辑DataGrid → 双击修改 → 保存写回SHP。支持添加字段和撤销修改。

![](screenshots/08-attr-manage.png)

## 9. 属性查询

选图层 → 选字段 → 选运算符(= != LIKE > <) → 输入值 → 执行查询 → 导出结果。

![](screenshots/09-attr-query.png)

## 10. 空间查询

选目标图层 + 源图层 → 选查询方式(相交/包含) → 执行空间查询。

![](screenshots/10-spatial-query.png)

## 11. 专题制图

选图层 → 输入图名 → 刷新底图 → 画布上出现地图底图+图例+比例尺+指北针 → 鼠标拖动各元素到想要位置 → 生成PNG。

图例读取图层真实渲染器，比例尺根据当前地图比例计算，指北针黑白样式。

![](screenshots/11-thematic.png)

## 12. 统计图查看

列出输出目录中的PNG统计图，缩略图预览，点击打开原图。

![](screenshots/12-stat-chart.png)

## 13. 统计表查看

列出输出目录中的CSV统计表，点击在DataGrid中预览内容。

![](screenshots/13-stat-table.png)

## 14. 视图菜单

标题栏"视图"下拉，切换面板显隐：数据面板、地图视图、图层面板、符号系统、地理处理。

![](screenshots/14-view-menu.png)

## 15. 状态栏

左侧显示操作状态和图层数，中间显示实时比例尺，右侧显示实时鼠标坐标（经纬度）。

比例尺从ArcGIS Runtime ViewpointChanged事件获取，坐标从MouseMove获取(100ms节流)。

![](screenshots/15-status-bar.png)
