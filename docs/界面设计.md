## 字体

全系统 Microsoft YaHei UI，标题 11pt，正文 10pt，表格 9pt。

## 启动界面

深色背景(#1B1F2B)，居中显示系统全称，底部三个圆点依次闪烁。

![](screenshots/ui-01.png)

## 主窗口布局

```
标题栏 (左:undo/redo+视图 | 右:最小化/最大化/关闭)
Ribbon (数据管理 | 土壤植被 | 专题制图)
┌────────┬──────────────┬────────┐
│数据面板 │    地图       │图层面板│
│        │              │符号面板│
│        │              │地理面板│
└────────┴──────────────┴────────┘
状态栏 (状态 | 比例尺 | 坐标)
```

![](screenshots/ui-02.png)

## Ribbon

每个Tab内容区为WrapPanel，按钮垂直排列(图标在上，文字在下)，56px高。每组底部有分组标签。

按钮hover时浅蓝底+蓝灰边框。

![](screenshots/ui-03.png)

## 数据面板

顶部：连接文件夹按钮 + 路径显示 + 刷新按钮
中部：绿色"自动生成属性表"按钮
搜索框
切换标签(空间数据管理/属性数据管理)
目录树：文件夹可展开，SHP/TIF文件高亮

![](screenshots/ui-04.png)

## 图层面板

每行：复选框 + 图层名 + 符号预览色块。悬停浅蓝底，选中深蓝左边框。

子级显示符号预览(小色块)，点击打开符号面板。

![](screenshots/ui-05.png)

## 符号面板

上半部分：预览区 + 属性编辑(填充色/透明度/轮廓色/线宽)
下半部分：按字段配色(字段选择+色带选择+应用按钮)

![](screenshots/ui-06.png)

## 地理分析面板

图层选择区(分类面/灾害点) → 分类字段 → 分级分区(方法+分类数) → 分析方法 → 输出目录
运行/保存按钮
结果区：柱状图 + CF汇总条 + DataGrid表格

![](screenshots/ui-07.png)

## 专题制图面板

图层选择 + 图名输入
画布预览区(白底+灰边框)：地图底图 + 可拖动图例 + 可拖动比例尺 + 可拖动指北针
操作按钮：刷新底图 + 生成专题图 + 状态提示

![](screenshots/ui-08.png)

## 属性浏览/编辑/查询面板

属性浏览：图层选择 → DataGrid只读 → 复制/导出CSV
属性编辑：图层选择 → DataGrid可编辑 → 保存/撤销/添加字段
属性查询：图层选择 → 字段选择 → 条件输入 → 执行查询

![](screenshots/ui-09.png)

## 控件样式

ComboBox：白底灰边框 → hover蓝边框 → 下拉带阴影
DataGrid：浅灰表头 + 交替行色 + hover浅蓝 + 选中蓝底
ScrollBar：6px细条灰色，hover加深
TextBox：白底灰框 → 聚焦蓝框

![](screenshots/ui-10.png)

## 右键菜单

数据面板(SHP文件)：添加地图、缩放至图层、属性浏览、属性
图层面板(图层)：缩放至图层、符号系统、属性浏览(矢)、上移、下移、移除

![](screenshots/ui-11.png)
