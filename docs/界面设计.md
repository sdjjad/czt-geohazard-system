# 界面设计

截图放在 screenshots/ 目录下。

---

## 色彩

| 用途 | 色值 |
|------|------|
| 主题色 | #1565C0 |
| 选中边框 | #0D47A1 |
| 悬停 | #BBDEFB |
| 选中背景 | #E3F2FD |

## 字体

全系统 Microsoft YaHei UI，标题 11pt，正文 10pt，表格 9pt。

## 启动界面

深色背景(#1B1F2B)，居中显示系统全称，底部三个圆点依次闪烁。

![](screenshots/ui-01.png)

## 主窗口布局

标题栏、Ribbon三页签、左侧数据面板、中央地图、右侧图层面板，底部状态栏。

![](screenshots/ui-02.png)

## 数据面板

顶部连接文件夹按钮和路径显示，绿色生成属性表按钮，搜索框，空间/属性切换标签，目录树。

![](screenshots/ui-03.png)

## 图层面板

每行含复选框、图层名、符号预览色块。悬停浅蓝底，选中深蓝边框。

![](screenshots/ui-04.png)

## 符号面板

上半部分预览和属性编辑，下半部分按字段配色。

![](screenshots/ui-05.png)

## 地理分析面板

图层选择、字段选择、分级分区、分析方法、输出目录、运行按钮。结果区含柱状图、CF汇总、DataGrid表格。

![](screenshots/ui-06.png)

## 专题制图面板

图层选择、图名输入、画布预览（地图底图+可拖动图例+比例尺+指北针）、生成按钮。

![](screenshots/ui-07.png)

## 属性浏览 / 编辑 / 查询

属性浏览：选择图层查看属性表。属性编辑：可编辑DataGrid、保存、撤销。属性查询：条件查询。

![](screenshots/ui-08.png)

## 空间查询

目标图层和源图层选择，相交/包含查询方式。

![](screenshots/ui-09.png)

## 统计图 / 统计表

查看已生成的PNG统计图和CSV统计表。

![](screenshots/ui-10.png)

## 右键菜单

数据面板右键：添加地图、缩放、属性浏览、属性。图层面板右键：缩放、符号、属性浏览、排序、移除。

![](screenshots/ui-11.png)

## 视图菜单

标题栏视图下拉，切换面板和地图视图的显隐。

![](screenshots/ui-12.png)

## 状态栏

左侧操作状态和图层数，中间实时比例尺，右侧实时鼠标经纬度坐标。

![](screenshots/ui-13.png)
