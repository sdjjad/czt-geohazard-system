# 系统架构

## 概述

WPF .NET 10 桌面应用，ArcGIS Runtime 地图引擎，面向长株潭地区地质灾害与土壤植被关系分析。

## 技术栈

| 层 | 技术 | 备注 |
|----|------|------|
| 运行时 | .NET 10.0 | WPF workload |
| 地图 | ArcGIS Runtime 200.6.0 | SHP/TIF加载、空间分析、符号渲染 |
| 面板 | AvalonDock 4.74.1 | 可停靠面板 |
| 几何 | NetTopologySuite 2.6.0 | 辅助计算 |

## 架构分层

```
Views (表现层)
  MainWindow, SplashWindow, MapView, GeoProcessToolView,
  ThematicMapToolView, AttributeTableView 等

Services (服务层)
  GeoAnalysisService, MapLayerService, ChartImageService,
  ThematicMapService, AttributeTableGenerator

Models (模型层)
  GeoAnalysisModels, SpatialDataModels, SymbolModels

Engine (引擎)
  ArcGIS Runtime WPF, WPF DrawingVisual
```

## 启动流程

```
App.OnStartup → SplashWindow(3.2秒) → MainWindow(最大化)
  MainWindow构造:
    InitializeComponent (加载XAML)
    加载数据目录树
    初始化MapView (ArcGIS Runtime地图, WGS84, 白底)
    初始化MapLayerService (图层集合, 绑定LayerTreeView)
    隐藏符号面板和地理处理面板
    注册坐标/比例尺状态栏事件
```

## 窗口结构

```
标题栏 (undo/redo + 视图菜单 + 窗口控制)
Ribbon (数据管理 / 土壤植被 / 专题制图)
DockingManager
  ├── 左: 数据面板 (DataPanelAnchor)
  ├── 中: 地图 (MapView)
  └── 右: 图层面板 / 符号面板 / 地理处理面板
状态栏 (图层数 / 比例尺 / 坐标)
```

## Ribbon 路由

所有按钮 Click → OpenTool_Click → 读 Button.Tag → switch 分发

```
数据管理 Tab:
  ImportData       → ImportDataDialog     (导入文件对话框)
  SpatialQuery     → OpenSpatialQuery     (空间查询面板)
  SpatialAnalysis  → OpenGeoTool          (地理分析面板)
  Mapping          → OpenThematicMap      (专题制图面板)
  AttributeBrowse  → OpenAttributeTable   (属性浏览面板)
  AttributeQuery   → OpenAttributeQuery   (属性查询面板)
  AttributeManage  → OpenAttributeManage  (属性编辑面板)

土壤植被 Tab (全部 → OpenGeoTool):
  SoilTypeAnalysis / SoilMoisture / VegType / VegCoverage / NDVI

专题制图 Tab:
  StatChart   → OpenStatChartViewer   (查看统计图)
  StatTable   → OpenStatTableViewer   (查看统计表)
  ThematicMap → OpenThematicMap       (专题制图面板)
```

## 面板管理

```
DataPanelAnchor    默认显示
LayerPanelAnchor   默认显示
SymbolPanelAnchor  默认隐藏, 点击图层名或符号时 Show()
GeoPanelAnchor     默认隐藏, 点击分析/制图按钮时 Show()

隐藏面板弹出时自动前置: IsActive=true, IsSelected=true
```

## 性能

- AllowsTransparency=False (透明窗口会强制软件渲染)
- App启动时 RenderOptions.ProcessRenderMode=Default 强制GPU
- MouseMove坐标采集100ms节流 (ScreenToLocation是重操作)
- 无 DropShadowEffect (位图特效触发软件渲染)
