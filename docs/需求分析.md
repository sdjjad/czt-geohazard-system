## 1. 项目概况

### 1.1 项目背景

湖南省长株潭地区（长沙、株洲、湘潭）地处江南丘陵地带，地质构造复杂，降雨充沛，是滑坡、崩塌、泥石流等地质灾害的高发区。土壤类型与植被覆盖是滑坡灾害发育的关键影响因子——不同土壤类型的工程地质性质（渗透性、抗剪强度等）差异显著，植被通过根系固土、截留降雨等机制影响斜坡稳定性。

本项目以 **土壤植被因子** 为核心，构建一套面向长株潭地区地质灾害评估的专题地理信息系统（Mini ArcGIS Pro），实现数据管理—分析计算—成果输出全流程自动化。

### 1.4 项目目标

- 下载并整编长株潭地区的土壤类型（2-3级分类）、土壤湿度、植被类型（2-3级分类）、植被覆盖度、NDVI 等指标数据
- 对各指标进行科学合理的分级分区
- 统计分析各地质灾害要素在不同类型区中的灾害数量、密度、百分比、CF值
- 制作灾害分布图、统计图、统计表
- 开发原型 Demo：用户通过简单选择参数，点击"确定"即可完成全流程处理，结果以专题图/统计图/表等形式自动保存

### 1.6 术语定义

| 术语 | 定义 |
|------|------|
| **CF值 (Certainty Factor)** | 确定性系数法，评价各因子类别对地质灾害发生的敏感程度。CF ∈ [-1, 1]：正值有利于灾害发生，负值不利于，0表示不相关 |
| **信息量法** | 基于信息论的评价方法，计算各因子类别提供的信息量 I = ln((Ni/N)/(Si/S)) |
| **证据权法** | 基于贝叶斯统计的二元评价方法，计算各证据因子的正负权重 W⁺、W⁻ |
| **层次分析法 (AHP)** | 多准则决策方法，通过专家打分构造成对比较矩阵计算因子权重 |
| **NDVI** | 归一化植被指数，反映植被生长状态和覆盖度的遥感指标，取值 [-1, 1] |
| **2-3级分类** | 土壤/植被分类体系中的中高级分类单元（如土类→亚类→土属） |

#### NS图2：CF值计算算法

```
┌──────────────────────────────────────────────────────────────────┐
│              确定性系数法 (CF值) 计算 NS 图                         │
├──────────────────────────────────────────────────────────────────┤
│  输入：研究区总面积 S_total，灾害点总数 N_total                      │
├──────────────────────────────────────────────────────────────────┤
│  计算先验概率 PP_s = N_total / S_total                             │
├──────────────────────────────────────────────────────────────────┤
│  FOR EACH 参数 p IN 用户选择的参数列表                              │
├──────────────────────────────────────────────────────────────────┤
│    获取参数 p 的分级类别列表 Classes[]                              │
├──────────────────────────────────────────────────────────────────┤
│    FOR EACH 类别 c IN Classes                                     │
├──────────────────────────────────────────────────────────────────┤
│      计算类别 c 的面积 S_i                                         │
├──────────────────────────────────────────────────────────────────┤
│      空间叠加：统计类别 c 内的灾害点数量 N_i                         │
├──────────────────────────────────────────────────────────────────┤
│      计算 PP_a = N_i / S_i      // 类别内灾害密度                  │
├──────────────────────────────────────────────────────────────────┤
│      ┌──────────────────────────────────────────────────┐        │
│      │  IF PP_a >= PP_s                                  │        │
│      │  ┌─────────────────┐                             │        │
│      │  │ CF = (PP_a-PP_s) │                             │        │
│      │  │    / (PP_a*(1-PP_s))                           │        │
│      │  └─────────────────┘                             │        │
│      │  ELSE                                             │        │
│      │  ┌─────────────────┐                             │        │
│      │  │ CF = (PP_a-PP_s) │                             │        │
│      │  │    / (PP_s*(1-PP_a))                           │        │
│      │  └─────────────────┘                             │        │
│      └──────────────────────────────────────────────────┘        │
├──────────────────────────────────────────────────────────────────┤
│      密度 Density_i = N_i / S_i                                   │
│      百分比 Percentage_i = N_i / N_total × 100%                   │
├──────────────────────────────────────────────────────────────────┤
│      记录 StatResult: 参数名, 类别名, N_i, S_i, Density, %, CF     │
├──────────────────────────────────────────────────────────────────┤
│    END FOR                                                        │
├──────────────────────────────────────────────────────────────────┤
│  END FOR                                                          │
├──────────────────────────────────────────────────────────────────┤
│  返回 List<StatResult>                                            │
└──────────────────────────────────────────────────────────────────┘
```

### 3.4 业务规则

| 编号 | 规则描述 |
|------|---------|
| BR-01 | 分析前必须至少加载一个灾害点图层和一个指标数据图层 |
| BR-02 | 指标数据的分级分类应参照国家标准（如GB/T 17296-2009《中国土壤分类与代码》） |
| BR-03 | CF值计算必须基于灾害点在各分级类别中的面积占比 |
| BR-04 | 输出目录不存在时自动创建；输出文件已存在时自动追加时间戳避免覆盖 |
| BR-05 | 所有分析结果需记录元数据（参数、方法、时间、数据源） |

### 3.5 核心计算公式

#### CF值（确定性系数）

$$CF = \begin{cases} \frac{PP_a - PP_s}{PP_a(1 - PP_s)} & PP_a \geq PP_s \\[10pt] \frac{PP_a - PP_s}{PP_s(1 - PP_a)} & PP_a < PP_s \end{cases}$$

其中：$PP_a$ = 某分类中灾害点面积占该分类总面积的比例；$PP_s$ = 研究区灾害点总面积占研究区总面积的比例

#### 信息量

$$I = \ln\frac{N_i / N}{S_i / S}$$

#### 灾害密度与百分比

$$D_i = \frac{N_i}{S_i} \qquad P_i = \frac{N_i}{N} \times 100\%$$

### 9.2 文献案例 — 三峡库区滑坡易发性评价

- 研究者：王某某等（2020），《基于CF-信息量模型的三峡库区滑坡易发性评价》
- 指标体系：坡度、坡向、高程、地层岩性、**土壤类型**、**植被覆盖度(NDVI)**、距河流距离、距道路距离
- 方法：CF值法 + 信息量法 + 逻辑回归
- **结论**：土壤类型和NDVI在评价模型中权重分别为0.15和0.12，位列第2和第4重要因子
